# Installation instructions

Install Gradle [here](https://gradle.org/install/). 

Next, open up the terminal and type (assuming Linux)

```
$ ./gradlew run
```

On Windows, open `cmd` and run :

```
gradlew.bat run
```

Ensure that Java JDK is installed besides gradle.

Admin username is `admin` and password is `admin`.

All the documentation is in the "docs" directory.
